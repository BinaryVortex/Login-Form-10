# Login Form (Login-Form-10)

A clean, responsive login form built using plain HTML and CSS.

![Preview](./Screenshot%202024-05-28%20005709.png)

## Overview

This repository contains a single-page login form created with semantic HTML and modern CSS. It's designed as a UI component / template you can reuse in small projects, demos, or to practice front-end skills.

## Features

- Simple, minimalist design
- Responsive layout that adapts to common screen sizes
- Pure HTML and CSS — no JavaScript required
- Single file structure for easy reuse (`index.html`)

## Preview

See the screenshot above for a quick preview of the form.

## Usage

1. Clone or download the repository.

   git clone https://github.com/BinaryVortex/Login-Form-10.git

2. Open `index.html` in your browser (double-click the file or serve it with a static server).

3. Customize the HTML/CSS as needed.

Optional: To preview changes during development you can use a simple static server such as Live Server in VS Code or Python's built-in server:

- Python 3: `python -m http.server 8000`
- Then open http://localhost:8000 in your browser.

## File structure

- `index.html` — Main file containing the markup and styles for the login form
- `Screenshot 2024-05-28 005709.png` — Preview image used in this README

## Customization

Edit the HTML markup and CSS inside `index.html` to change colors, fonts, or layout. The structure is intentionally simple to make customization easy.
